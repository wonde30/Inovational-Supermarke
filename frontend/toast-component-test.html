Accessibility Features

  • Icons: Each toast type has a unique icon (checkmark, X, warning triangle, info circle)
  • ARIA roles: role="alert" for screen reader announcements
  • ARIA live regions: aria-live="assertive" for errors, "polite" for others
  • Screen reader labels: Hidden text labels ("Success:", "Error:", etc.)
  • Close button labels: Descriptive aria-label for close buttons
  • Color contrast: White text on colored backgrounds meets WCAG AA standards
