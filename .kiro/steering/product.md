# Product Overview

**IBMS (Inventory and Billing Management System)** - A comprehensive smart supermarket management platform with integrated storefront capabilities.

## Core Functionality

- **Inventory Management**: Product catalog, categories, stock tracking, reorder alerts, expiry monitoring
- **Point of Sale (POS)**: Sales processing, real-time stock validation, multi-payment gateway support
- **Order Management**: Customer orders, delivery tracking, order status workflows
- **Purchase Orders**: Supplier management, purchase order lifecycle, stock confirmation
- **Storefront**: Public-facing e-commerce with cart, checkout, QR code scanning
- **Analytics & Reports**: Sales reports, profit analysis, cashier performance, stock alerts, audit logs
- **Multi-language Support**: English, Amharic, Oromo with user-specific language preferences
- **Role-Based Access**: Admin, Manager, Cashier, Customer, Delivery Staff, Supplier roles

## Payment Integration

- Stripe
- PayPal
- Chapa (Ethiopian payment gateway)

## Key Features

- Email verification and password reset
- Social authentication (Google OAuth)
- QR code product scanning
- Dark mode support
- Real-time stock alerts (low stock, out of stock, expiring soon, expired)
- Automated daily summaries
- AI chatbot assistance
- Theme customization
